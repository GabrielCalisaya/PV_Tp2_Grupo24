Podemos cambiar los estilos de una página web utilizando un botón mediante JavaScript. Para hacerlo, se define un botón en el HTML y se le asigna una función que se ejecutará cuando el usuario haga clic. Esta función puede cambiar propiedades CSS como el color de fondo, el tamaño de fuente o el tipo de letra. Por ejemplo, al hacer clic en un botón, podemos cambiar el fondo de la página a un color diferente o aumentar el tamaño del texto. Esto permite que la página sea más interactiva y se adapte a las preferencias del usuario.
